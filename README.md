# musicPlayer

ㅁ 개요

- 뮤직 플레이어의 모양만 갖춤. 특별한 기능은 없으며, 인터랙티브 강의를 보고 실습한 내용을 정리한 파일
- 페이지 이전과 다음 버튼을 누르면 다음 화면으로 이동 가능
- 모바일 환경에 맞춰서 터치 스타트와 터치 엔드를 이용하여 페이지 이동 가능
